# 🎓 Student Management System

A responsive and interactive web-based Student Management System developed using HTML, CSS, and JavaScript. This application allows users to efficiently manage student records with features like adding, editing, and deleting student details.

---

## 🚀 Features

- ➕ Add new student details
- 📋 View student list dynamically
- ✏️ Edit existing student records
- ❌ Delete student with confirmation popup
- 💾 Data handled using JavaScript (DOM manipulation)
- 🎨 Clean and user-friendly UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
